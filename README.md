## Prerequisites

$ Ruby: 2.5.8$

$ Rails: 5.2.6$

$ Postgres: 9.5$

## How to use

Clone Repository
```
$ git clone https://github.com/SunnySparks/Lifestyle-articles.git

```

Access your repository by typing in your terminal
```
$ cd lifestyle-articles

```

Install dependencies
```
$ bundle install

```

Setup database with:
```
$ rails db:create
rails db:migrate

```

Run the app
```
$ rails server

```

Open `http://localhost:3000/` in your browser.

Run tests
```
$ rspec

```
